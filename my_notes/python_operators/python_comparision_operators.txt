            PYTHON COMPARISION OPERATORS

==      Equal
!=      Not Equal
>       Greater than
<       Less than
>=      Greater than or or Equal to
<=      Less than or Equal to



>>> b = 4
>>> c = 3
>>> print(b == c)
False
>>> print(b!= c)
True
>>> print(b > c)
True
>>> print(b < c)
False
>>> print(b >= c)
True
>>> print(b <= c)
False
